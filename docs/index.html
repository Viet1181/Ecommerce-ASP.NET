Tổng quan về ứng dụng

Ứng dụng Thương mại Điện tử ASP.NET MVC là một hệ thống bán hàng trực tuyến đầy đủ tính năng, bao gồm cả phần người dùng và phần quản trị. Ứng dụng được xây dựng trên nền tảng ASP.NET MVC 5, sử dụng Entity Framework để tương tác với cơ sở dữ liệu SQL Server.

Ứng dụng cung cấp các chức năng chính như:

  • Quản lý sản phẩm, danh mục, thương hiệu
  • Hệ thống giỏ hàng và thanh toán
  • Quản lý đơn hàng
  • Quản lý người dùng và phân quyền
  • Tìm kiếm và lọc sản phẩm

Cài đặt và cấu hình

Yêu cầu hệ thống

  • Visual Studio 2019 trở lên
  • SQL Server 2014 trở lên
  • .NET Framework 4.7.2 trở lên
  • IIS Express hoặc IIS

Các bước cài đặt

  1. Clone Repository 
    git clone https://github.com/username/e-commerce-mvc.git
  2. Mở Solution

    Mở file NguyenQuocViet_2122110285.sln bằng Visual Studio

  3. Khôi phục Database

    Sử dụng SQL Server Management Studio để tạo cơ sở dữ liệu mới và chạy script ECommece.sql để tạo các bảng và dữ liệu mẫu

  4. Cấu hình Connection String

    Mở file Web.config và cập nhật connection string để kết nối với cơ sở dữ liệu

    <connectionStrings>
    <add name="ECommerceDBEntities" connectionString="metadata=res://*/Context.dbModel.csdl|res://*/Context.dbModel.ssdl|res://*/Context.dbModel.msl;provider=System.Data.SqlClient;provider connection string="data source=YOUR_SERVER;initial catalog=YOUR_DATABASE;integrated security=True;MultipleActiveResultSets=True;App=EntityFramework"" providerName="System.Data.EntityClient" />
</connectionStrings>
  5. Build và chạy ứng dụng

    Nhấn F5 hoặc Ctrl+F5 để build và chạy ứng dụng

Kiến trúc hệ thống

Ứng dụng được xây dựng theo mô hình MVC (Model-View-Controller), theo đó:

  • Model: Đại diện cho dữ liệu và logic nghiệp vụ
  • View: Hiển thị dữ liệu và giao diện người dùng
  • Controller: Xử lý các request từ người dùng, tương tác với Model và trả về View

Chức năng chính

Phần người dùng

  • Đăng ký/Đăng nhập: Người dùng có thể đăng ký tài khoản mới và đăng nhập vào hệ thống
  • Xem sản phẩm: Hiển thị sản phẩm theo danh mục, thương hiệu
  • Tìm kiếm: Tìm kiếm sản phẩm theo từ khóa và danh mục
  • Giỏ hàng: Thêm sản phẩm vào giỏ hàng, cập nhật số lượng hoặc xóa sản phẩm
  • Đặt hàng: Đặt hàng và theo dõi tình trạng đơn hàng
  • Xem lịch sử đơn hàng: Người dùng có thể xem lại các đơn hàng đã đặt

Phần quản trị

  • Quản lý sản phẩm: Thêm, sửa, xóa sản phẩm và ảnh sản phẩm
  • Quản lý danh mục: Thêm, sửa, xóa danh mục sản phẩm
  • Quản lý thương hiệu: Thêm, sửa, xóa thương hiệu
  • Quản lý đơn hàng: Xem, cập nhật trạng thái đơn hàng
  • Quản lý người dùng: Quản lý thông tin người dùng

Cơ sở dữ liệu

Ứng dụng sử dụng Entity Framework với mô hình Database First để tương tác với cơ sở dữ liệu SQL Server. Các bảng chính trong cơ sở dữ liệu bao gồm:

  • Users: Lưu trữ thông tin người dùng
  • Categories: Lưu trữ thông tin danh mục sản phẩm
  • Brands: Lưu trữ thông tin thương hiệu
  • Products: Lưu trữ thông tin sản phẩm
  • ProductImages: Lưu trữ ảnh sản phẩm
  • Orders: Lưu trữ thông tin đơn hàng
  • OrderDetails: Lưu trữ chi tiết đơn hàng
  • Carts: Lưu trữ thông tin giỏ hàng
  • Reviews: Lưu trữ đánh giá sản phẩm
  • Wishlists: Lưu trữ danh sách yêu thích

API và Integration

Ứng dụng cung cấp một số API để tích hợp với các hệ thống khác. Để sử dụng API, bạn cần có API Key và cấu hình phù hợp.

Các API chính

  • Sản phẩm API: Cho phép truy vấn và quản lý sản phẩm
  • Đơn hàng API: Cho phép tạo và quản lý đơn hàng
  • Người dùng API: Cho phép quản lý thông tin người dùng

Triển khai

Triển khai trên IIS

  1. Mở Internet Information Services (IIS) Manager
  2. Tạo một website mới hoặc application mới
  3. Chọn đường dẫn đến thư mục chứa ứng dụng
  4. Cấu hình application pool sử dụng .NET Framework 4.7.2 hoặc cao hơn
  5. Cấu hình connection string trong Web.config
  6. Khởi động ứng dụng

Triển khai trên Azure

  1. Tạo một Azure App Service
  2. Cấu hình deployment từ Git repository
  3. Tạo một Azure SQL Database
  4. Cập nhật connection string trong Application Settings của Azure App Service
  5. Chạy migrations để tạo cơ sở dữ liệu

Câu hỏi thường gặp

Để reset mật khẩu admin, bạn cần truy cập vào cơ sở dữ liệu và cập nhật mật khẩu trong bảng Users. Lưu ý rằng mật khẩu được mã hóa bằng MD5, bạn cần cung cấp giá trị MD5 mới.

Để thêm phương thức thanh toán mới, bạn cần:
  1. Tạo một class mới trong thư mục Payment để xử lý logic thanh toán
  2. Cập nhật interface IPaymentMethod
  3. Thêm phương thức thanh toán mới vào OrderController
  4. Thêm view mới để hiển thị form thanh toán

Để cấu hình email thông báo, bạn cần cập nhật các thông tin SMTP trong Web.config: 
<system.net>
    <mailSettings>
        <smtp from="your-email@example.com">
            <network host="smtp.example.com" port="587" userName="your-email@example.com" password="your-password" enableSsl="true" />
        </smtp>
    </mailSettings>
</system.net>
